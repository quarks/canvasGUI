# canvasGUI   3.0.0 
### (c) Peter Lager  2026 
A JavaScript library to create a GUI (Graphical User Interface) directly on an HTML canvas element.

Compatible with any JacaScript application and [p5js](https://p5js.org).
On-canvas GUI library for p5js

For more information about using canvasGUI please look at
* canvasGUI [website](http://www.lagers.org.uk/canvasgui) for detailed coverage of this library, its controls and usage.
* canvasGUI [API reference](http://www.lagers.org.uk/canvasgui/ref/)

Additional guides covering aspects of the API and controls are in the *Guides* folder.
